
Converting Hexadecimal to Decimal :
------------------------------------
1) int x4 = 0x234;

  (234) Base16 = (?) Base10

 = 4*16power0 + 3*16power1 + 2*16power2

 = 4*1 + 3*16 + 2*256

 = 564


2)  int x5 = 0733;

   (733) Base8 = (?) Base10

   = 3*8power0 + 3*8power1 + 7*8power2

   = 3*1 + 3*8 + 7 * 64

   = 3 + 24 + 448

   = 475
